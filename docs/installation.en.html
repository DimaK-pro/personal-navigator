Installing Personal Navigator Skill

If you are not familiar with GitHub or agent skills, start here. You need to get the project folder, copy personal-navigator-skill/ into the skills location used by your agent environment, restart the agent, and begin the first conversation.

What to do after installation Full Markdown guide
Important: run the commands below from the root of the downloaded personal-navigator repository. Private user memory files should be created in your active workspace, not inside personal-navigator-skill/.
01

Download

Get the repository through Git or Download ZIP.

02

Copy the skill

Move personal-navigator-skill/ into your environment's skills folder.

03

Restart

Restart the agent so it discovers the new skill.

04

Start the Map

Ask the agent to guide a friendly interview.

1. Get the project

If Git is installed, use:

git clone https://github.com/DimaK-pro/personal-navigator.git
cd personal-navigator

If Git is not installed: open the repository from the header, click Code, choose Download ZIP, unzip it, and open the personal-navigator folder in your agent environment.

2. Choose your environment

Codex

Personal install across projects.

mkdir -p ~/.agents/skills
cp -R personal-navigator-skill ~/.agents/skills/

Antigravity

Workspace install inside the current project.

mkdir -p .agents/skills
cp -R personal-navigator-skill .agents/skills/

Claude Code

Personal install.

mkdir -p ~/.claude/skills
cp -R personal-navigator-skill ~/.claude/skills/

Claude Code project

Install only for the current project.

mkdir -p .claude/skills
cp -R personal-navigator-skill .claude/skills/

OpenCode

Project skill.

mkdir -p .opencode/skills
cp -R personal-navigator-skill .opencode/skills/

Windsurf / Cascade

Workspace skill.

mkdir -p .windsurf/skills
cp -R personal-navigator-skill .windsurf/skills/

GitHub Copilot / VS Code

Project skill for the repository.

mkdir -p .github/skills
cp -R personal-navigator-skill .github/skills/

Cursor, Gemini CLI, Aider

Use AGENTS.md as a bridge and ask the agent to read the skill from this repository.

Read AGENTS.md and use Personal Navigator.
Start with personal-navigator-skill/SKILL.md and guide me through the Personality Map.

3. Restart the agent

After copying the folder, close and reopen Codex, Antigravity, Claude Code, or your chosen environment. This lets the agent rescan available skills.

4. First prompt after installation

Use $personal-navigator-skill.
I want to start building my Personality Map. Guide me through a friendly interview.

Expected behavior: the agent explains that full navigation begins only after the Map, creates or offers memory files, and asks the first grounded human question.

5. If the agent cannot write files

Use $personal-navigator-skill.
You cannot write files automatically. Show me ready-to-save English text for NAVIGATOR_STATE.md, PERSONALITY_MAP.md, DEVELOPMENT_JOURNAL.md, and OPEN_LOOPS.md so I can save them manually.

6. Verify installation

Use Personal Navigator.
I do not have a Personality Map yet. Start a friendly interview with me and create memory files if you can.

If it works, the agent will not make deep conclusions without data, will mark the limitation before a full Map exists, and will start building the 9 blocks through a calm interview.
