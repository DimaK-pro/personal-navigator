Установка Personal Navigator Skill

Если вы не погружены в GitHub и agent skills, начните здесь. Вам нужно получить папку проекта, скопировать папку personal-navigator-skill/ в место, где ваша агентная среда ищет skills, перезапустить агента и начать первый диалог.

Что делать после установки Полная Markdown-инструкция
Важно: команды ниже выполняются из корня скачанного репозитория personal-navigator. Личные файлы памяти пользователя создаются в вашем рабочем пространстве, а не внутри personal-navigator-skill/.
01

Скачать

Получите репозиторий через Git или Download ZIP.

02

Скопировать skill

Перенесите personal-navigator-skill/ в папку skills вашей среды.

03

Перезапустить

Перезапустите агента, чтобы он увидел новый skill.

04

Начать Карту

Попросите агента вести дружеское интервью.

1. Получите проект

Если у вас установлен Git, используйте команды:

git clone https://github.com/DimaK-pro/personal-navigator.git
cd personal-navigator

Если Git не установлен: откройте репозиторий из шапки, нажмите Code, выберите Download ZIP, распакуйте архив и откройте папку personal-navigator в своей агентной среде.

2. Выберите вашу среду

Codex

Личная установка для всех проектов.

mkdir -p ~/.agents/skills
cp -R personal-navigator-skill ~/.agents/skills/

Antigravity

Workspace-установка внутри текущего проекта.

mkdir -p .agents/skills
cp -R personal-navigator-skill .agents/skills/

Claude Code

Личная установка.

mkdir -p ~/.claude/skills
cp -R personal-navigator-skill ~/.claude/skills/

Claude Code project

Установка только внутри текущего проекта.

mkdir -p .claude/skills
cp -R personal-navigator-skill .claude/skills/

OpenCode

Project skill.

mkdir -p .opencode/skills
cp -R personal-navigator-skill .opencode/skills/

Windsurf / Cascade

Workspace skill.

mkdir -p .windsurf/skills
cp -R personal-navigator-skill .windsurf/skills/

GitHub Copilot / VS Code

Project skill для репозитория.

mkdir -p .github/skills
cp -R personal-navigator-skill .github/skills/

Cursor, Gemini CLI, Aider

Используйте AGENTS.md как мост и попросите агента читать skill из репозитория.

Read AGENTS.md and use Personal Navigator.
Start with personal-navigator-skill/SKILL.md and guide me through the Personality Map.

3. Перезапустите агента

После копирования закройте и снова откройте Codex, Antigravity, Claude Code или другую среду. Это нужно, чтобы агент заново просканировал папки skills.

4. Первый запрос после установки

Use $personal-navigator-skill.
Хочу начать собирать мою Карту личности. Веди меня через дружеское интервью.

Ожидаемое поведение: агент объяснит, что полная навигация начинается только после Карты, создаст или предложит создать файлы памяти и задаст первый человеческий вопрос.

5. Если агент не может сам писать файлы

Use $personal-navigator-skill.
Ты не можешь записывать файлы автоматически. Покажи мне на русском готовые тексты для NAVIGATOR_STATE.md, PERSONALITY_MAP.md, DEVELOPMENT_JOURNAL.md и OPEN_LOOPS.md, чтобы я сохранил их вручную.

6. Проверка установки

Use Personal Navigator.
У меня пока нет Карты личности. Начни со мной дружеское интервью и создай файлы памяти, если можешь.

Если всё работает, агент не будет делать глубокие выводы без данных, отметит ограничение до полной Карты и начнёт сбор 9 блоков через спокойное интервью.
