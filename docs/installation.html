Установка Personal Navigator Skill

Personal Navigator работает как переносимый agent skill: папка с `SKILL.md`, reference-файлами и шаблонами памяти. Полная навигация начинается после создания 9-блочной Карты личности.

Главное правило: внутреннее ядро skill описано на английском, но пользовательские ответы, интервью, Карта, журнал, open loops и готовые блоки памяти создаются на языке пользователя.

Что устанавливается

personal-navigator-skill/
  SKILL.md
  agents/openai.yaml
  references/
  templates/

Codex

mkdir -p ~/.agents/skills
cp -R personal-navigator-skill ~/.agents/skills/

После установки перезапустите Codex и начните диалог так:

Use $personal-navigator-skill.
Хочу начать собирать мою Карту личности. Веди меня через дружеское интервью.

Универсальный workspace-путь

Для сред с поддержкой открытого формата `SKILL.md` используйте:

mkdir -p .agents/skills
cp -R personal-navigator-skill .agents/skills/

Antigravity

Workspace: `.agents/skills/`. Для глобальной CLI-установки сверяйтесь с актуальной документацией Antigravity.

Claude Code

Project: `.claude/skills/`. Personal: `~/.claude/skills/`.

OpenCode

Project: `.opencode/skills/`. Global: `~/.config/opencode/skills/`.

Windsurf / Cascade

Workspace: `.windsurf/skills/`. Global: `~/.codeium/windsurf/skills/`.

GitHub Copilot / VS Code

Project skills можно хранить в `.github/skills/`, `.claude/skills/` или `.agents/skills/`.

Cursor, Gemini CLI, Aider

Используйте `AGENTS.md` как мост: укажите агенту читать `personal-navigator-skill/SKILL.md` и нужные references.

Проверка установки

Use Personal Navigator.
У меня пока нет Карты личности. Начни со мной дружеское интервью и создай файлы памяти, если можешь.

Ожидаемое поведение: агент говорит, что без Карты полная навигация невозможна, создаёт или предлагает 4 файла памяти, задаёт первый живой вопрос и не делает глубоких выводов без данных.
