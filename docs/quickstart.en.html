Quickstart

After installation, you do not need to understand the whole methodology first. Ask the agent to use Personal Navigator and start with the Personality Map.

1. Start the first conversation

Use $personal-navigator-skill.
I want to start building my Personality Map. Guide me through a friendly interview.
Expected agent behavior:

It explains that full personal navigation is unavailable without a Map, creates or offers memory files, and asks the first grounded human question.

2. Check memory files

The user's workspace should contain:

NAVIGATOR_STATE.md
PERSONALITY_MAP.md
DEVELOPMENT_JOURNAL.md
OPEN_LOOPS.md
supplements/

This is private user memory. It should not be stored inside `personal-navigator-skill/`.

3. Answer in ordinary language

The Navigator should ask one strong question at a time. A good answer does not need to sound polished; it should be honest and grounded in real situations, choices, consequences, and energy.

4. If help is needed before the full Map

Use $personal-navigator-skill.
I do not have a Map yet, but I need to understand my current situation. Give limited navigation and ask the missing questions.

Correct behavior: the agent helps, but clearly says the answer is provisional because the 9 Map blocks are not complete yet.

5. When Map V1 is ready

The agent should complete all 9 blocks, run an integration pass, remove duplicates, move uncertainty into `OPEN_LOOPS.md`, and only then announce the first full Map version.

Answers, questions, the Map, journal, and open loops should be in the user's language. English reference files are the internal core, not the user interface.
Personal Navigator Skill. Start with the Map, then navigate from it.
