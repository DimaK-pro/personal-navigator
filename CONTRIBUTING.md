# Contributing

Personal Navigator is an open-source skill for building a living Personality
Map, development journal, open loops, and grounded personal navigation.

Good contributions include:

- bug reports where the skill over-personalizes, guesses, or ignores missing
  Map context;
- examples where the interview flow feels too heavy, too vague, or too
  mechanical;
- improvements to language behavior, memory templates, safety boundaries, or
  installation instructions;
- realistic scenarios that help validate the skill across languages and agent
  environments.

When reporting an issue, include:

- what you expected the Navigator to do;
- what it actually did;
- whether a full Personality Map existed;
- the relevant lifecycle state, if known: `NO_MAP`, `MAP_BUILDING`, or
  `NAVIGATION`;
- a minimal prompt or memory excerpt if it is safe to share.

Do not share private Personality Maps, journal entries, or sensitive personal
data publicly. Redact names, places, and details before opening an issue.

The project welcomes practical feedback from people who use the skill in real
self-navigation work. The goal is to make the methodology clearer, safer, more
grounded, and more useful.
